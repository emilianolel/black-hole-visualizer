# 📜 Project Utility Scripts

A collection of global scripts for environment setup, data ingestion, and project-wide automation.

## 📂 Structure
-   `setup/`: Initial environment and dependency installers. 🛠️
-   `dev/`: Development environment helpers (e.g., Jupyter connectivity). 💻
-   `data/`: Data pipeline automation and BigQuery ingestion. 🗄️
-   `infra/`: Wrapper scripts for Terraform management. 🏗️
