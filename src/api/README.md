# 🌉 Data API

A robust **FastAPI** backend that bridges the gap between BigQuery and the React frontend.

## ✨ Features
-   **BigQuery Integration**: Efficiently queries massive datasets of billions of traced photons. 📊
-   **Smart Caching**: In-memory optimization to handle frequent viewing angles.
-   **FastAPI**: Asynchronous performance for low-latency data streaming. 🐍
-   **Secure Access**: Built-in authentication for research data protection. 🔐
